Markdown Knowledge Graph (`mdkg`) is git-native project memory for AI-assisted software engineering.

This Starlight site is the future canonical documentation host for `docs.mdkg.dev`. The existing Markdown files at the top of `docs/` remain committed source and compatibility inputs for the current docs checks while the Starlight pages are brought online.

## Start here

- Install the CLI and initialize a graph.
- Learn the core project-memory concepts.
- Use goals, tasks, tests, checkpoints, packs, and handoffs to make AI-assisted work auditable.
- Keep secrets, raw prompts, provider payloads, and bulky runtime traces out of graph state.

## Launch boundary

These docs describe current public-alpha capabilities and planned launch surfaces. They do not claim hosted queues, worker execution, public event/reducer/lease/materializer CLI surfaces, or comprehensive secret scanning.
