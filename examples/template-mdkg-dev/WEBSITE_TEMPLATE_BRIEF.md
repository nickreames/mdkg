# Website Template Brief

Generate a candidate mdkg.dev-inspired static site from this cloned graph. The site should be inspectable, fast, conservative in claims, and easy to discard or promote after review.

## Required Story Beats

- start from `goal-1`
- use `spike-1` to choose audience, visual direction, and page scope
- build a candidate site locally
- validate with `test-1`
- record whether the candidate should be discarded, previewed, or promoted later

## Boundaries

- no production deploy by default
- no analytics activation
- no DNS changes
- no credentials
- no raw prompt transcript storage
