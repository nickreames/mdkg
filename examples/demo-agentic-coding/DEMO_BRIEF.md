# Demo Brief

Build a small inspectable static artifact that shows how an agent can start from an mdkg goal, gather context, implement a scoped change, run checks, and write a checkpoint. The output should be understandable in a live talk without private data.

## Required Story Beats

- start from `goal-1`
- use `spike-1` to decide audience and proof path
- build the smallest useful local artifact
- validate with `test-1`
- close with `chk-1` or a follow-up checkpoint

## Boundaries

- no live deploy
- no credentials
- no raw prompt transcript storage
- no dependency on the parent mdkg repo beyond the published CLI behavior
