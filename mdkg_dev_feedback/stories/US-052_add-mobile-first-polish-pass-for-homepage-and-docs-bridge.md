**Priority:** P1
**Theme:** Responsive UX

- [ ] Hero headline wraps cleanly on narrow screens.
- [ ] CTA buttons stack or wrap without cramped layout.

- [ ] Feature cards stack in a sensible order.
- [ ] Footer links are tap-friendly.
- [ ] Docs bridge page is useful on mobile.
